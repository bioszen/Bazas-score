# Bazas Score

Static GitHub Pages version of Bazas Score.

## Files

- `index.html`: the app entry point for GitHub Pages.
- `manifest.webmanifest`: PWA metadata for installable browsers.
- `service-worker.js`: offline cache for the app shell.
- `icons/`: app icons for browser tabs, Android, and iOS home screen shortcuts.
- `.nojekyll`: tells GitHub Pages to publish files as plain static assets.

## GitHub Pages

After pushing this folder to a GitHub repository, enable GitHub Pages from the repository settings and publish from the default branch root.

The app stores each match locally in the user's browser. It does not sync games between devices.
